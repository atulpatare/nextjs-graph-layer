This repo demonstrate the use of `deck.gl graph layers` to display a custom layer of rounded rectangle.

## Getting Started

- Pre-requistic

```
node version 18
```

- Clone the repo

```
git clone git@github.com:atulpatare/nextjs-graph-layer.git
```

- Get the dependencies

```
npm install
```

- Run

```
npm run dev
```
